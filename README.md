# mujoco_basic
Basic MuJoCo simulator and rendering example

# Requirements
* OpenCV 4 or higher
* OpenGL
* Pthreads
* MuJoCo (included through fetch)
* GLFW3 (included through fetch)
* Dear ImGui (included through fetch)

# Build
To build use the ```build.sh``` script.

```bash
./build.sh
```

# Run
To run use the ```run.sh``` script.

```bash
./run.sh
```
